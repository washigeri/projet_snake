Detailed Description

Ce source contient l'implementation des fonctions gerant les fonctions prncipales du jeu.

Les fonctions dans ce fichier permettent notamment la creation du plateau de jeu, la detection des entrees utilisateurs et le deplacement des serpents. Ce fichier implemente aussi les deux types d'IA: idle et defensive.
